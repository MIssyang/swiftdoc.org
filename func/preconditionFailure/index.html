@noreturn func preconditionFailure(_ message: @autoclosure () -> String = default, file: StaticString = default, line: UWord = default)

A fatal error occurred and program execution should stop in debug mode and in optimized mode. In unchecked builds this is a noop, but the optimizer can still assume that the call is unreachable.
